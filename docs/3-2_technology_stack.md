# Технологический стек

## Программное обеспечение для проектирования

- **Figma** - cервис для создания макетов интерфейса, кнопок, страниц сайта, логотипов и иконок
- **Draw.io** - создаёт схемы и диаграммы (ER-диаграммы базы данных, схемы процессов и т.д)

## Програмное обеспечение для разработки

### Бэкенд (серверная часть, логика сайта)

- **Python**
- **Django** — основной фреймворк (маршрутизация, модели, формы, шаблоны).
- **Django REST Framework (DRF)** — создание REST API для ботов (в будущем).
- **Django Allauth** — регистрация и авторизация пользователей (мастера и клиенты).
- **Celery + Redis** — фоновая обработка задач (уведомления, напоминания о записи).
- **django-crispy-forms / django-bootstrap5** — оформление форм (запись, отзывы, регистрация).
- **django-filter** — фильтры для каталога услуг и мастеров.
- **Pillow / django-imagekit** — обработка изображений мастеров и портфолио.
- **django-ckeditor** — редактор контента для мастеров и описаний услуг.

### Фронтэнд

- **HTML / CSS / JavaScript** — базовая разметка и стили.
- **Tailwind CSS** — современная и быстрая стилизация интерфейса.
- **Alpine.js** — интерактивные элементы (лайки, модальные окна, фильтры).

### Базы данных

- **PostgreSQL**
- **Redis** — кеш и очередь задач Celery.
- **S3‑совместимое облако (Yandex Cloud / AWS S3)** - хранение медиа

### Инфраструктура

- **Docker** — контейнеризация проекта.
- **Gunicorn + Nginx** — продакшн-сервер.
- **Git + GitHub** — контроль версий и деплой.
- **Let’s Encrypt / Certbot** — SSL‑сертификаты для HTTPS.

### Тестирование и отладка

- **pytest + pytest-django** — тестирование логики моделей и форм.
- **Playwright** — end-to-end тестирование функционала сайта.
- **Postman** — тестирование REST API.

### Аналитика и мониторинг

- **Prometheus** — сбор метрик работы сайта.
- **Grafana** — визуализация метрик.



## Примечания по структуре проекта

- **Личный кабинет мастера** — добавление работ, управление портфолио, просмотр записей.
- **Личный кабинет клиента** — история записей, повторная запись, уведомления о записи.
- **Онлайн-запись** — выбор услуги **→** мастера **→** времени **→** подтверждение.
- ⚡️**Боты (Telegram)** — интеграция через API в будущем.
